- Slightly more body text per slide is acceptable (LinkedIn audience expects depth)
- Square format — all content must work within a 1:1 canvas

## LAYOUT SYSTEM

Each slide internal structure:
- Padding: 40px all sides
- Top zone (first 80px): eyebrow label left-aligned + slide counter right-aligned
- Mid zone: headline — left-aligned, strong visual weight
- Lower zone: body copy and any supporting visual element
- Bottom strip (last 20px): thin border line #27272A

The 540×540px canvas must be fully utilized. No empty lower halves.

Slide variation rules — no two consecutive slides feel identical:
1. Full headline dominant — headline takes 60% of canvas, minimal body below
2. Editorial split — headline top-left, structured element bottom-right
3. Analytical — headline + mechanism visual (flow, list, status indicators)
4. Close — centered, gold headline, strong final statement
