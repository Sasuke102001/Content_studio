Responsive Behavior: At breakpoint below 768px, heading sizes scale down to maintain visual hierarchy on smaller screens while preserving readability. Body (16px), Small (14px), Caption (12px), and Label (12px) remain unchanged — they are already optimized for mobile. All line heights and letter spacing remain consistent across breakpoints.

Implementation Notes

Clash Display (Headings): Used exclusively for H1–H4 and Display. Creates premium brand presence and clear visual hierarchy. Maintains intentional, strong typography consistent with brand doctrine. Negative letter-spacing on larger sizes creates tighter, more premium appearance.
Inter (Body & UI): Default sans-serif for all body copy, UI labels, captions, and supporting text. Optimized for readability and scanning. Open spacing and moderate weights ensure clarity on dark backgrounds.
Line Height Strategy: Larger sizes use tighter line height (1.1–1.2) for impact; body text uses generous spacing (1.5–1.6) for readability and reduced visual density. This creates natural hierarchy without requiring size changes alone.
Letter Spacing: Headline sizes use negative tracking to create premium compression. Mid-range text uses neutral spacing. UI labels use increased spacing (+0.8px) for clarity and formality. This creates visual rhythm across the scale.
Dark Background Optimization: All styles tested and optimized for dark (#0A0A0A) primary background. Text colors use #F5F5F5 (primary) and #A1A1AA (secondary) for sufficient contrast and reduced eye strain. Generous line heights support readability without additional weight.
Minimum Size Rule: Never use Body or smaller text below 16px in body copy, 14px in supporting text, or 12px in labels. Smaller sizes compromise readability and accessibility.
Mobile Responsive Strategy: At breakpoint <768px, heading sizes scale proportionally to maintain hierarchy while preventing overwhelming headline sizes on mobile screens. Use CSS media queries to apply mobile sizes:

@media (max-width: 767px) {
h1 { font-size: 36px; }
h2 { font-size: 28px; }
h3 { font-size: 22px; }
h4 { font-size: 18px; }
.lead { font-size: 18px; }
}
