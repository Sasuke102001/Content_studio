Layout Dimensions

Max Content Width
Desktop
1200px
Maximum content width constraint. Centered container with side padding (80px each). Total viewport width = 1200px + (80px × 2) = 1360px.
Horizontal Padding
Desktop (≥768px)
80px (left & right)
Premium page padding at desktop sizes. Creates breathing room and premium positioning. Uses 4xl spacing token.
Mobile (<768px)
24px (left & right)
Responsive reduction for mobile screens. Maintains content readability while optimizing screen real estate. Uses 6x spacing (24px = 6 × 4px).

Usage Guidelines

Component Spacing Hierarchy

Tight Spacing
xs (4px) + sm (8px)
Used within components, between form inputs, inline element gaps. Creates compact, dense layouts.
Component Padding
md (16px) + lg (24px)
Card padding, section internal padding, component breathing room. Creates visual grouping.
Section Spacing
xl (32px) + 2xl (48px)
Between major content sections, large component gaps, page vertical rhythm.
Page-Level Spacing
3xl (60px) + 4xl (80px)
Hero section padding, major section breaks, page margins. Creates macro-level hierarchy.
4px Grid Discipline: Every spacing value is a multiple of 4. This creates alignment consistency across all layouts. Never use arbitrary spacing values like 5px, 10px, 15px, or 18px. The grid is the foundation of visual rhythm.
Responsive Behavior: The spacing scale remains consistent across breakpoints. Only horizontal padding changes (80px → 24px at mobile). All other spacing values stay fixed to maintain visual hierarchy and rhythm.
Content Width Constraint: Max content width of 1200px keeps reading lines at optimal length and prevents layouts from becoming too wide. This applies to all text content, card grids, and content-heavy sections.
Gap vs Padding: Use gap property for gutters between grid/flex items. Use padding for whitespace inside containers. This keeps spacing concerns separated and makes responsive adjustments cleaner.

Implementation Examples

CSS Variables (Copy to stylesheets)
:root {
  --space-xs: 4px;
  --space-sm: 8px;
  --space-md: 16px;
  --space-lg: 24px;
  --space-xl: 32px;
  --space-2xl: 48px;
  --space-3xl: 60px;
  --space-4xl: 80px;
  
  --max-width: 1200px;
  --padding-desktop: 80px;
  --padding-mobile: 24px;
}

@media (max-width: 767px) {
  body {
    padding-left: var(--padding-mobile);
    padding-right: var(--padding-mobile);
  }
}
Container Pattern
.container {
  max-width: var(--max-width);
  margin: 0 auto;
  padding: 0 var(--padding-desktop);
}

@media (max-width: 767px) {
  .container {
    padding: 0 var(--padding-mobile);
  }
}
Card with Content Pattern
.card {
  padding: var(--space-md);
  background: var(--bg-card);
  border-radius: 8px;
}

.card-header {
  margin-bottom: var(--space-lg);
}

.card-items {
  display: flex;
  flex-direction: column;
  gap: var(--space-sm);
}
Section Pattern
section {
  padding: var(--space-3xl) 0;
  border-bottom: 1px solid var(--border-muted);
}

section + section {
  margin-top: var(--space-xl);
}

.section-title {
  margin-bottom: var(--space-2xl);
}
