## FORMAT
This is a LinkedIn document carousel.
Each slide:
- Width: 540px
- Height: 540px (1:1 square ratio — LinkedIn standard)
- Total slides: 7
- All slides rendered as a single scrollable HTML page (stacked vertically)
- Each slide uses class="slide"

LinkedIn document carousels display slightly differently from Instagram:
- Slightly more body text per slide is acceptable (LinkedIn audience expects depth)
- Square format — all content must work within a 1:1 canvas
- Professional, editorial, analytical tone throughout

## LAYOUT SYSTEM
Each slide internal structure:
- Padding: 40px all sides
- Top zone (first 80px): eyebrow label left-aligned + slide counter right-aligned
- Mid zone: headline — left-aligned, strong visual weight
- Lower zone: body copy and any supporting visual element
- Bottom strip (last 20px): thin border line #27272A

The 540×540px canvas must be fully utilized. No empty lower halves.
Slide variation rules — no two consecutive slides feel identical:
1. Full headline dominant — headline takes 60% of canvas, minimal body below
2. Editorial split — headline top-left, structured element bottom-right
3. Analytical — headline + mechanism visual (flow, list, status indicators)
4. Close — centered, gold headline, strong final statement

HARD RULE — NO EMPTY CONTAINERS:
Never output a <div> or container that has no text content, no <svg>, and no
background-image/gradient/pattern. Every element in the lower zone must either
(a) contain real text or numbers, or (b) be one of the Visual Component Library
patterns above, fully populated with content. Thin 1-2px horizontal/vertical rule
lines (the bottom strip, gold rules, section separators) are exempt — those are
intentional structural lines, not placeholders. If a slide's content doesn't
naturally fill the canvas, use a smaller, more centered headline-dominant layout
(variation rule 1 or 4) rather than adding a decorative empty space-filler.
